# Source Ledger

Last updated: 2026-05-08

## Purpose

This ledger maps important claims to source URLs, local evidence paths, and practical implications. Use it when a future run needs the reason behind a decision without rereading every research note.

## How To Use

- Start here when choosing research, structure, automation, or eval work.
- Treat `docs/research/*.md` and `logs/*.md` as the dereferenceable evidence layer.
- Add a row when a source materially changes the operating loop, automation prompt, eval plan, or memory structure.
- If a claim becomes obsolete, mark it as stale instead of silently deleting it.
